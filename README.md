SQL Project

Projeto desenvolvido durante o Bootcamp de QA da TripleTen.

Objetivo

Realizar consultas em banco de dados para validação de informações, investigação de inconsistências e apoio à análise de defeitos.

Atividades Realizadas

* Consultas SQL
* Filtragem de dados
* Agrupamentos
* Validação de resultados

Ferramentas Utilizadas

* SQL
* Google Docs

Resultado

Execução de consultas para análise e validação de dados, contribuindo para a identificação de inconsistências e verificação da qualidade das informações armazenadas no banco de dados.
